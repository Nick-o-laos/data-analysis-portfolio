# Motodynamiki Data Analysis Project

## Project Overview
Statistical analysis performed for the course "Advanced Data Analysis with R" at Athens University of Economics and Business. The project analyzes client data from Motodynamiki to identify characteristics of customers who purchase on-desk increments.

## Files
- `analysis/RMarkdown_code.Rmd` - Complete R Markdown analysis code
- `analysis/Report.pdf` - Detailed project report
- `analysis/Presentation.pptx` - Presentation slides
- `README.md` - This file

## Key Analysis Steps
1. Data cleaning and preparation
2. Exploratory Data Analysis (EDA)
3. Pairwise comparisons and correlation analysis
4. Multiple regression modeling
5. Model interpretation

## Technologies Used
- R (with packages: psych, dplyr, glmnet, olsrr, etc.)
- R Markdown
- Statistical analysis techniques
